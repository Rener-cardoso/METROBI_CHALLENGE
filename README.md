## Getting Started

To run the application locally, follow these steps:

1. Clone the repository:

```bash
  git clone git@github.com:Rener-cardoso/METROBI_CHALLENGE.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```
